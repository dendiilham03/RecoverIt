### RecoverIt

RecoverIt helps abuse the Windows service failure recovery function to execute a payload. This allows us to avoid the issue of having to change the service's ImagePath.

### Command Line Syntax

**RecoverIt.exe `<ServiceName`> `<ProgramPath`> `<Arguments`>**
